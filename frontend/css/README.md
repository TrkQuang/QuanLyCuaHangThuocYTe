# CSS (Styling)

Chứa các file CSS để trang trí giao diện HTML.

## Chức năng:

- Định dạng màu sắc, font chữ
- Layout responsive
- Style cho form, button, table
- Animation và transition

## Các file CSS:

- `style.css` - Style chung
- `form.css` - Style cho form
- `table.css` - Style cho bảng
- `responsive.css` - Style Responsive design
