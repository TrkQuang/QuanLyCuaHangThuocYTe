# Resources

Chứa các file tài nguyên và cấu hình cho ứng dụng.

## Chức năng:

- File cấu hình database (db.properties)
- File cấu hình ứng dụng (application.properties)
- File SQL scripts
- File log configuration
- Static resources (nếu cần)

## Ví dụ:

```properties
# db.properties
db.url=jdbc:mysql://localhost:3306/quanlythuoc
db.username=root
db.password=your_password
db.driver=com.mysql.cj.jdbc.Driver
```
